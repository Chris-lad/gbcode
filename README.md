http://www.ulb.ac.be/di/map/gbonte/mod_stoch/syl.pdf



To use it in your R code

- library(devtools)

- install_github("gbonte/gbcode")

- require(gbcode)


To change the directory to the one containing scripts 

- setwd(find.package("gbcode"))
